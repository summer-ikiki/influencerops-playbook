# InfluencerOps Demo

A lightweight static demo of an AI-assisted influencer marketing workflow dashboard.

## What it shows

- Gmail-style outreach queue with batch send and follow-up rules
- Creator evaluation based on 14-day view median, school signal, comment quality, and estimated CPM
- Duplicate check and approval workflow before outreach confirmation
- Creator pipeline across screening, approval, contract, content review, and posted stages
- Content schedule with post-link tracking
- Payment status updates with confirmation behavior
- Campaign reporting with creator ranking, actual vs estimated metric review, and AI-style takeaways
- Long-term partner tracker for repeat creator collaborations

## Run locally

Open `index.html` in a browser for the playbook. Open `dashboard.html` for the interactive dashboard demo. No build step or server is required.
